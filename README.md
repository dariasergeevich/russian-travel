# Проект: Путешествие по России

### Описание проекта

Проект сделан на основе макета Figma для устройств с разрешением 1280px, 1024px, 768px и 320px.

### Используемые технологии
* Адаптивная верстка
* Grid и flex
* Медиа-запросы
